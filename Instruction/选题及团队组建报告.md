# 选题及团队组建报告

## 一、选题基本信息

| 项目 | 内容 |
|------|------|
| **项目名称** | 学习养成计划系统 |
| **选题编号** | 2.5 |
| **选题来源** | 软件工程课程设计题目库 |
| **项目简介** | 开发一个学习养成计划软件，帮助用户添加、管理学习任务，提供提醒功能、复习提醒、任务分析和可视化图表展示。 |

| **作业时间** | 2026-07-06 11:32:00 至 2026-07-06 23:59:00 |

### 核心功能模块

1. **学习任务模块**：用户自定义模块和系统推荐任务模块。
2. **任务属性设置**：主题、完成时间、优先级，以及是否需要复习提醒。
3. **任务提醒功能**：根据设定的完成时间和优先级对用户进行不同频率的提醒。
4. **打卡任务模块**：用户完成任务后，系统会生成打卡记录。
5. **数据展示**：以进度条、统计表和打卡日历等方式展示用户打卡记录。
6. **任务分析功能**：针对用户每天的任务添加数和完成率进行统计。

## 四、项目目标

> 项目目标引用自 项目要求.md：
>
> 开发一个学习养成计划软件，帮助用户添加、管理学习任务，提供提醒功能、复习提醒、任务分析和可视化图表展示。功能涵盖学习任务模块（用户自定义模块和系统推荐任务模块）、任务属性设置（主题、完成时间、优先级及复习提醒）、任务提醒功能（根据完成时间和优先级进行不同频率提醒）、打卡任务模块（完成任务后生成打卡记录）、数据展示（进度条、统计表、打卡日历）以及任务分析功能（每日任务添加数和完成率统计）。
